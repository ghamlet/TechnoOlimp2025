1 урок
1.Что такое Generative AI?
A Тип ИИ для принятия решений.
B Тип ИИ, создающий новый контент на основе паттернов.
C Тип ИИ только для числовых данных.
D Тип ИИ исключительно для текста.
Правильный ответ: Тип ИИ, создающий новый контент на основе паттернов.


2.В чем заключается основная функция весов и смещения в нейронной сети?
A Определить скорость работы нейронной сети.
B Хранить данные для нейронной сети.
C Контролировать силу связей и корректировать выходные данные.
D Решать, какие входные данные игнорировать.
Правильный ответ: Контролировать силу связей и корректировать выходные данные.

3.Какой критический шаг следует после прямого прохода (forward pass) при обучении нейронной сети?
A Корректировка смещения.
B Вычисление потерь (loss).
C Инициализация весов.
D Аугментация данных.
Правильный ответ: Вычисление потерь

4.Какое значительное достижение в области нейронных сетей было представлено исследователями Google в 2017 году?
A Сверточные нейронные сети (CNN).
B Рекуррентные нейронные сети (RNN).
C Архитектура Transformer.
D Диффузионные модели.
Правильные ответ: Архитектура Transformer



2 урок
1.Что означает аббревиатура LLM?
А Большие светящиеся модели (Large Luminous Models)
В Большие языковые модели (Large Language Models)
С Ни один из вышеперечисленных.
Правильный ответ: Большие языковые модели (Large Language Models)

2.Почему их называют «большими» (large)?
A Они содержат миллиарды параметров.
B Они занимают много физического пространства (примерно как кенгуру).
Правильный ответ: Они содержат миллиарды параметров.

3.Что делают модели (в контексте машинного обучения)?
A Принимают входные данные и выдают значения, например, прогнозы.
B Стоят на месте, чтобы их нельзя было увидеть.
C Ничего.
Правильный ответ:  Принимают входные данные и выдают значения, например, прогнозы.


3 урок
1.Что означает аббревиатура RAG в контексте ИИ?
A Random Answer Generator
B Retrieval-Augmented Generation
C Rapid Algorithm Growth
D Регулярная автономная генерация
Правильный ответ: Retrieval-Augmented Generation

2.Какой основной недостаток чистых LLM (больших языковых моделей) решает RAG?
A Низкая скорость работы
B Галлюцинации (ложные ответы) 
C Ограниченное количество языков
D Высокая стоимость обучения
Правильный ответ: Галлюцинации (ложные ответы)

3. Какие два основных этапа включает процесс RAG?
A Токенизация и генерация
B Поиск информации и её генерация
C Обучение и тестирование
D Сжатие данных и шифрование
Правильный ответ: Поиск информации и её генерация


4 урок
1. Что такое Runway ML
A Игровой движок
B Платформа для AI-создания медиа
C Социальная сеть
D Язык программирования
Правильный ответ: Платформа для AI-создания медиа

2. Какие типы задач можно решать с помощью Runway ML?
A Только генерация изображений
B Генерация изображений, редактирование видео, обработка аудио
C Программирование нейросетей с нуля
D Создание 3D-анимаций
Правильный ответ: Генерация изображений, редактирование видео, обработка аудио

3. Как называется функция Runway ML для создания изображений по текстовому описанию?
A Text-to-Image
B Image-to-Text
C Style Transfer
D Audio Cleanup
Правильный ответ: Text-to-Image
